# Gemini Chatbot

A simple chatbot application using **Google Gemini API**, powered by **FastAPI** for the backend and **Streamlit** for the frontend. This chatbot allows users to interact with the Gemini model in real-time.

---

## Features

- FastAPI backend serving a `/chat` endpoint
- Streamlit frontend for interactive chatting
- Supports conversation history
- Uses **Google Gemini 2.5 Flash** model
- Easy environment setup with `.env` configuration

---

## Requirements

- Python 3.10+
- pip (Python package manager)
- Google API Key for Gemini
